Plugin allows you to add 301 redirects for site pages.

Features:
- ease of use
- the ability to add a redirect to any place in the table
- adding several redirects at once
- Validation column "From"
- protection against cyclic redirects
- possibility of using "*" as in regular expressions(bulk redirect)

== Installation ==

1. Upload the `TE Redirect` folder to the `/wp-content/plugins` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the `301 Redirect`
